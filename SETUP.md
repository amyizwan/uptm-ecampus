# UPTM eCampus Setup Guide

## Quick Start:
1. Create MySQL database: uptm_ecampus
2. Run: php artisan migrate
3. Run: php artisan db:seed --class=SampleDataSeeder

## Test Logins:
- Student: student@uptm.edu.my / password
- Lecturer: lecturer@uptm.edu.my / password  
- Admin: admin@uptm.edu.my / password

## Features Included:
✅ Student Dashboard (Grades, Attendance, Documents)  
✅ Lecturer Dashboard (Assignment Management, Grading)  
✅ Announcements System  
✅ Assignments & Submissions  
✅ User Authentication

## Access URLs:
- Main Site: http://localhost:8000
- Student: http://localhost:8000/student
- Lecturer: http://localhost:8000/lecturer
- Admin: http://localhost:8000/admin
